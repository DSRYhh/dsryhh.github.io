SphereFace: Deep Hypersphere Embedding for Face Recognition

Paper: https://arxiv.org/abs/1704.08063

Introduction

This paper propose a new loss function to do face recognition, which called A-Softmax (Angular Softmax). Paper modified softmax and add a angular parameter $m$.

As most loss funtion in face recognition, A-softmax can learn discriminative face features. The final purpose is minimal interclass distance is larger than maximal intra-class distance. With proposed parameter $m$, we can control the margin betwenn different classes.

Basic idea
